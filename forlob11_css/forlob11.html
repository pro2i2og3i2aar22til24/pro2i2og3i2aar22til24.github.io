Forløb 11 : Introduktion til css

I dette lille forløb skal i øve jer i hvordan man laver css ved at lave et lille css der gør denne webside noget mere spændende at se på.

Hvis dette er meget nemt og i bliver hurtigt færdige skal i lave prøve at ændre lectios forside vha. et style-sheets værktøj.

Ting i skal lære:

  • CSS lav styling af bestemte "class", "id" eller "tags"
  • CSS lav styling af bestemte kombinationer af class/id/tags
  • CSS indsæt baggrundsbilleder, style af billedet, og hent det via en url

Værtøj til customs css:

  • stylbot for chrome

Opgave 1:

Først skal i prøve at lave formatere dette webside. Jeg forslår i kopierer html-kildekoden over i på jeres egen lokal webserver og prøver at lege med det derfra.

Opgave 2:

Fri leg med Lectios forside. Download stylbot - og anvend den til at lave jeres eget jule-tema.
