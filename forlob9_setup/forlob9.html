Inden i skal begynde at lave kode html er det nødvendigt at i får installerer nogle værktøjer. Se nedenfor.

Lektion 1 : Værktøjer og dit første "lokale" website, vscode og live-server Lektion 2 : Åben dit "lokale" website på flere enheder

2. Offentligt tilgængelig website vha. github pages

Github har et system kaldet github pages, hvor man gratis kan hoste hjemmesider, det skal vi prøve

  1. Opret konto på github pages med et bestemt brugernavn
  2. Opret et repository kaldet "brugernavn".github.io f.eks. hedder "ajrptec.github.io"
  3. Upload dit website, du skal omdøbe det til "index.html", til dit repository
  4. Gå ind under "Actions" på dit repository, her kan du se hvornår dit website er ffærdig deployed

3. Åben dit "lokale" website på flere enheder

Du kan sagtens nøjes med at se dit website via din egen computer. Men det kan jo også være sjovt at se det på andre enheder på samme tid.

  1. Del din internetforbindelse via computer eller mobil
  2. Forbind til internettet din ikke-dele-enhed til dele-enheden
  3. Find den delene enheds IP-adresse og live-serverens port
  4. Åben nu siden ved at skrive IP-adresse:port ....... F.eks. "192.168.1.16:5500"

4. Undersøg dit website i med "udviklingsværtøjet" direkte i din browser

Højre-klik på dit website i din browser og se/manipuler med "html" direkte. Kun til test formål.
